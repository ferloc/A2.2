Solucion de errores

Lo que haremos es cambiar los valores de yes y no por valores de 1 y 0 respectivamente. Y en pdays cambiar el valor de -1 a 0 para que sea un no Adems de cambiar el nombre de la variable y a Salidapara evitar confusiones.

df.loc[df['pdays'] == -1, 'pdays'] = 0

La columna pdays representa el nmero de das que pasaron desde que el cliente fue contactado por ltima vez. El valor -1 en esta columna indica que el cliente no fue contactado previamente. Para facilitar el anlisis, reemplazamos -1 con 0, ya que 0 es un valor ms intuitivo para representar "ningn contacto previo". loc es una funcin de Pandas que permite acceder a un grupo de filas y columnas por etiquetas o una condicin booleana. En este caso, seleccionamos todas las filas donde pdays == -1 y asignamos el valor 0 a la columna pdays.

df.rename(columns={'y': 'Salida'}, inplace=True)

La columna y es la variable objetivo (target) que indica si el cliente suscribi un depsito a plazo (yes/no). Cambiamos su nombre a Salida para que sea ms descriptivo y evitar confusiones. inplace=True Este parmetro indica que el cambio se aplicar directamente al DataFrame (df) sin necesidad de asignar el resultado a una nueva variable.

df = pd.get_dummies(df, drop_first=True)

Las variables categricas (como job, marital, education, etc.) no pueden ser usadas directamente en modelos matemticos porque no son numricas. pd.get_dummies() convierte estas variables categricas en variables dummy (tambin llamadas one-hot encoding). Cada categora se convierte en una nueva columna binaria (0 o 1). Por ejemplo, si la columna job tiene las categoras management, technician, y retired, se crearn tres columnas: job_management, job_technician, y job_retired. drop_first=True Este parmetro elimina la primera categora de cada variable categrica para evitar la multicolinealidad (dependencia lineal entre variables). Por ejemplo, si job tiene 3 categoras, solo se crearn 2 columnas dummy.

df = df.astype(int)

Despus de crear las variables dummy, las columnas resultantes son de tipo bool (True/False). Los modelos estadsticos y de machine learning requieren que los datos sean numricos, por lo que convertimos estas columnas booleanas a enteros (1 para True y 0 para False).

# Funcin para calcular mtricas de la matriz de confusin
def calcular_metricas(cm):
    TP = cm[1, 1]  # Verdaderos Positivos
    TN = cm[0, 0]  # Verdaderos Negativos
    FP = cm[0, 1]  # Falsos Positivos
    FN = cm[1, 0]  # Falsos Negativos

    # Exactitud
    exactitud = (TP + TN) / (TP + TN + FP + FN)
    
    # Sensibilidad (Recall)
    sensibilidad = TP / (TP + FN)
    
    # Especificidad
    especificidad = TN / (TN + FP)

    return exactitud, sensibilidad, especificidad

Vamos a podar el rbol de decisin determinando el valor ptimo de mediante validacin cruzada con 4 folds.

Importamos numpy y las funciones cross_val_score y StratifiedKFold de sklearn.model_selection. Usamos StratifiedKFold(n_splits=4) para asegurar una separacin estratificada en los folds. Creamos un vector de posibles valores de con numpy.linspace(0.001, 0.2, N), donde N est entre 100 y 250. Inicializamos un vector para almacenar los resultados de cada fold.

In[34]:
# Importar numpy
import numpy as np
# Importar funciones de validacin cruzada
from sklearn.model_selection import cross_val_score, StratifiedKFold
# Generar objeto de validacin cruzada
skf = StratifiedKFold(n_splits=4)
# Generar secuencia de valores
ccp = np.linspace(0.001, 0.2, 250)
# Inicializar variable de salida
cv_scores = []
# Ciclo donde se calcula el f1-score mediante validacin cruzada
for alpha in ccp:
  pruned_tree = DTC(ccp_alpha=alpha)
  cv_scores.append(np.mean(cross_val_score(pruned_tree, X_train, y_train, cv=skf, scoring='f1')))

Revisemos ahora qu valor de alpha es ptimo para este sistema. usando la funcin argmax de numpy en el vector de resultados, y ubicar dentro del vector de posibles $\alpha$ el valor que corresponde a dicha posicin.

In[37]:
# Encontrar el alpha ptimo
alpha = ccp[np.argmax(cv_scores)]
# Imprimir resultados
print("Best alpha:",alpha)
